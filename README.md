📊 Bank Loan Analysis Dashboard
Overview

This project analyzes bank loan data to evaluate loan performance, customer behavior, and lending trends. Using SQL and Power BI, the dashboard provides actionable insights into loan applications, funded amounts, repayments, and loan quality.


Objectives:-

Analyze loan application and approval trends.

Monitor key loan performance metrics.

Identify good and bad loans.

Understand customer demographics and loan purposes.

Support data-driven lending decisions.



Tools & Technologies:-

SQL – Data extraction and KPI calculations

Power BI – Dashboard creation and visualization

Power Query – Data cleaning and transformation

DAX – Advanced calculations and measures



Key Features:-

📈 Loan Applications Analysis

💰 Total Funded Amount & Amount Received Tracking

✅ Good Loan vs Bad Loan Performance

🌍 State-wise Loan Distribution

🏠 Home Ownership Analysis

🎯 Loan Purpose Insights

📊 Interactive Filters and Drill-down Visualizations


Key KPIs:-

Total Loan Applications

Total Funded Amount

Total Amount Received

Average Interest Rate

Average Debt-to-Income (DTI) Ratio

Good Loan Percentage

Bad Loan Percentage


Dashboard Insights:-

Identifies high-performing and risky loan segments.

Highlights regional lending patterns.

Tracks repayment performance across customer groups.

Supports strategic lending and risk management decisions.
